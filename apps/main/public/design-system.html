Components

BaseHeader

Shared header component with navigation, branding, and responsive behavior. Provides logo, navigation links, optional language switcher, and scroll-based shrinking animation.

Import: import { BaseHeader } from "@repo/ui"

Location: packages/ui/src/components/navigation/BaseHeader.tsx

Responsive:

  • Desktop (≥750px): Horizontal nav links with dropdown
  • Mobile (<750px): Hamburger menu with drawer from right

Scroll:

  • Header shrinks from 70px to 40px as user scrolls (configurable via shrinkOnScroll)
  • Optional background color transition after scroll threshold
Prop Type Default Description
backgroundColor string "transparent" Header background color
textColor string theme.palette.common.white Text/icon color
logoVariant "color" | "light" "light" Logo color variant
shrinkOnScroll boolean true Enable scroll-based shrinking
showLanguageSwitcher boolean false Show EN/ES language toggle
backgroundColorScrolled string - Color after scrolling past threshold
backgroundScrollThreshold number 200 Pixels to scroll before color change
onLogoClick () => void scroll to top Custom logo click handler

💡 Recommended: App-specific Header wrapper

Create an app-specific Header.tsx wrapper that imports BaseHeader and adds custom behavior (e.g., dynamic background based on scroll position, custom navigation). This keeps the base component reusable while allowing app-specific customization.

Example location: apps/main/app/components/Header.tsx

Basic Usage: 

  // Direct usage (simple apps)
  <BaseHeader showLanguageSwitcher />
  
  // With scroll-based background change
  <BaseHeader
    backgroundColor="transparent"
    backgroundColorScrolled="rgba(42, 82, 135, 0.75)"
    backgroundScrollThreshold={200}
  />

App-specific wrapper example: 

  // apps/main/app/components/Header.tsx
  import { BaseHeader } from "@repo/ui"
  import { useTabs } from "../context/Tabs"
  
  export function Header() {
    const { isInTabsArea } = useTabs()
  
    return (
      <BaseHeader
        backgroundColor={
          isInTabsArea ? "rgba(42, 82, 135, 0.75)" : "transparent"
        }
        onLogoClick={() => {
          // Custom smooth scroll logic
        }}
      />
    )
  }

WCAG 2.0 AA: Skip link, keyboard navigation, focus management, 44px touch targets, prefers-reduced-motion support, aria-expanded/haspopup/controls on menus.

Panel

Base full-viewport panel with standard spacing. Provides consistent outer spacing from theme tokens.

Import: import { Panel } from "@repo/ui"

Location: packages/ui/src/components/panels/Panel.tsx

Prop Type Default Description
backgroundColor string "transparent" Background color
backgroundImage string - Optional image URL (enables grid stacking)
fullHeight boolean true Use 100vh height
includePadding boolean true Apply standard panel spacing
id string - Panel ID for navigation
ariaLabel string - Accessible label for the section
sx SxProps - Additional MUI sx props

Usage Example: 

  <Panel
    id="intro"
    ariaLabel="Introduction"
    backgroundColor="#3D7DB5"
    sx={{
      display: "flex",
      flexDirection: "column",
      justifyContent: "space-between",
    }}
  >
    <Headline />
    <Body />
  </Panel>

Features: Semantic <section> element, WCAG compliant, supports background images via CSS grid stacking.
